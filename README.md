# AgendaApp Backend

Este é o backend do projeto **AgendaApp**, desenvolvido com **.NET 6**, utilizando **Clean Architecture** para gerenciamento de contatos (CRUD).

## Funcionalidades

- Endpoints RESTful para CRUD de contatos.
- Validação de dados com **FluentValidation**.
- Banco de dados **PostgreSQL** com **Entity Framework Core**.
- Documentação interativa com **Swagger**.

## Tecnologias Utilizadas

- **.NET 6**
- **Entity Framework Core**
- **FluentValidation**
- **AutoMapper**
- **PostgreSQL**
- **Swagger**

## Pré-requisitos

- **.NET SDK 6+**
- **PostgreSQL**
